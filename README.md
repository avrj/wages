# Prerequisites
* Leiningen
* Java

# Run project
`lein run`

# Run tests
`lein test`

# Build executable
`lein uberjar`

# Run executable
`java -jar target/uberjar/wages-0.1.0-SNAPSHOT-standalone.jar`
